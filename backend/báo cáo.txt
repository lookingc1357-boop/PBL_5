DỰ ÁN: IDE (Integrated Development Environment) - Phiên bản 0.0.1-SNAPSHOT


1. TỔNG QUAN DỰ ÁN
═══════════════════════════════════════════════════════════════════════════════

1.1 Mục tiêu chính:
   - Xây dựng IDE web-based hoạt động trên nền tảng cloud
   - Cung cấp khả năng quét phát hiện lỗ hổng bảo mật mã nguồn bằng AI
   - Hỗ trợ terminal tương tác qua Docker container
   - Quản lý dự án, file và workspace
   - Xác thực người dùng an toàn bằng JWT

1.2 Công nghệ chính:
   - Backend: Spring Boot 4.0.5 + Java 17
   - Database: MySQL
   - Messaging: WebSocket (STOMP) + Spring Messaging
   - Container: Docker Java API (3.7.1)
   - Terminal: PTY4J (0.13.12) - Pseudo Terminal
   - AI Model: CodeBERT / GraphCodeBERT qua HTTP
   - Bảo mật: JWT Authentication + Spring Security
   - Build: Maven

1.3 Kiến trúc hệ thống:
   ┌─────────────────┐
   │  Frontend (JS)  │
   │  (localhost:3000) │
   └────────┬────────┘
            │ WebSocket/HTTP
            ↓
   ┌─────────────────────────────────────────┐
   │     Spring Boot Backend (Này)           │
   │  - Authentication (JWT)                 │
   │  - WebSocket Message Broker (STOMP)    │
   │  - REST Controllers                     │
   │  - Services                             │
   └─────────┬──────────────────────────┬───┘
             │                          │
      ┌──────↓──────┐         ┌────────↓────────┐
      │   MySQL     │         │ Docker Engine   │
      │ Database    │         │ + Containers    │
      └─────────────┘         │ + PTY Terminal  │
                               └─────────────────┘
             ↓
      ┌─────────────┐
      │ AI Model    │
      │ (Port 8000) │
      └─────────────┘


2. NHỮNG KỀT QUẢ ĐÃ ĐẠT ĐƯỢC
═══════════════════════════════════════════════════════════════════════════════

2.1 CHỨC NĂNG HOÀN THÀNH:

   ✓ 2.1.1 XÁC THỰC & PHÉP BAN (Authentication & Authorization):
      - Hệ thống đăng nhập/đăng ký người dùng
      - Tạo JWT token với thời hạn 24 giờ (86400000ms)
      - Cookie HttpOnly + Secure + SameSite=None
      - Bảo mật: Stateless (STATELESS SessionCreationPolicy)
      - Mã hóa mật khẩu: BCrypt
      - Hỗ trợ CORS cho localhost:3000, *.run.app, *.trycloudflare.com
      
   ✓ 2.1.2 QUẢN LÝ DỰ ÁN & FILE (Project & File Management):
      - Tạo, xóa, liệt kê dự án
      - Lưu trữ tại: D:/docker_store/{username}/{projectname}
      - Hỗ trợ đọc ghi file (UTF-8 safe)
      - Copy, Move, Delete file/folder
      - Đọc metadata dự án (creation date, last modified, description)
      - Theo dõi thay đổi file theo thời gian thực (Watcher)
      
   ✓ 2.1.3 TERMINAL TƯƠNG TÁC (Interactive Terminal):
      - Khởi tạo PTY (Pseudo Terminal) qua PTY4J
      - Chạy shell bên trong Docker container (docker exec)
      - Đọc/ghi input-output từ shell một cách real-time
      - Quản lý nhiều terminal session cùng lúc (ConcurrentHashMap)
      - Gửi output về client qua WebSocket
      - Tự động thay đổi thư mục khi tạo terminal
      
   ✓ 2.1.4 CONTAINER MANAGEMENT (Docker Integration):
      - Tạo, khởi động, dừng, xóa Docker container
      - Kiểm tra trạng thái container (RUNNING/STOPPED)
      - Hỗ trợ volume mount (Docker volumes)
      - Cấu hình image: image:v1 (từ application.properties)
      - Liên kết container với người dùng (per-user container)
      
   ✓ 2.1.5 PHÂN TÍCH LỖ HỔng BẢO MẬT (Vulnerability Scanning):
      - Quét batch: Phân tích nhiều hàm cùng lúc
      - Quét single: Phân tích một hàm riêng lẻ
      - Giao tiếp với AI Model qua HTTP (WebClient reactive)
      - Trả về kết quả: Hàm, tên lỗi (CWE), độ nghiêm trọng
      - Async processing: Không block main thread (@Async)
      - Gửi kết quả qua WebSocket user queue
      
   ✓ 2.1.6 WEBSOCKET & REAL-TIME MESSAGING (STOMP Protocol):
      - Simple Message Broker (STOMP)
      - Prefix: /app (client → server), /queue (server → client)
      - Điểm kết nối: /ws
      - Hỗ trợ user-specific queue: /user/{username}/queue/*
      - Auto-subscribe khi client kết nối
      - Các endpoint:
        • /app/scan-vulnerability → Batch scan
        • /app/explorer/create → Tạo file/folder
        • /app/explorer/delete → Xóa file/folder
        • /app/explorer/read → Đọc file
        • /app/explorer/list → Liệt kê tree
        • /app/terminal/send → Gửi command
        
   ✓ 2.1.7 CẤU HÌNH VÀ CƠSỞ HẠ TẦNG:
      - Database MySQL (host: localhost:3306, db: PBL_5)
      - AI Model endpoint: http://localhost:8000
      - Hibernate DDL: update (auto-migrate schema)
      - Async executor: configurable thread pool
      - Configuration class: AsyncConfig, DockerConfig, WebSocketConfig, SecurityConfig
      
   ✓ 2.1.8 UNIT TEST:
      - Test framework: JUnit 5 + Mockito
      - Test case cho VulnerabilityService
      - Mock: AiModelClient, SimpMessagingTemplate
      - Kiểm tra Batch scan success scenarios


2.2 CẤU TRÚC CODE CHẤT LƯỢNG:
   - Sử dụng Lombok (@Slf4j, @RequiredArgsConstructor, @Getter, @AllArgsConstructor)
   - Dependency Injection (Spring IoC)
   - Separation of Concerns (Controller → Service → Repository)
   - DTOs rõ ràng cho data transfer
   - Async/concurrent processing cho I/O-bound tasks
   - Logging chi tiết (slf4j)
